How This Fits FUT Automation

Use FINTA to replace the manual Figma block-section-download-PDF step. Load metadata, choose sections, then export split PDFs for FUT Automation.

01

First load on a laptop

Enter title, Figma token, and file sRL. A successful load writes the saved entry to data/session_store.local.json.

02

Select sections

Choose a page, keep only the sections that represent FUT flows, and reorder them if needed from Full List.

03

Export PDF ZIP

Phase 4 renders checked sections as separate PDFs. The ZIP does not contain tokens.

04

Git safety

The local store is ignored by Git. Another laptop starts empty and must save its own Figma token once.
